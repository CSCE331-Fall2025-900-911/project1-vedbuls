Project: Family Tree Management App (CRUD Application) -

GitHub: CrudApp Repository

This project is a web application where users can manage a list of members and parent-child relationships. The app can perform all CRUD operations and assign parent-child relationships.

Features:

  • Add, Edit or Delete - Full CRUD operations to add, edit, or delete a table of people, given the properties of that member.
  • Generate a descendant tree for any selected person - If you click generate tree for any member, a tree with all descendants will display, showing all parent-child relationships within the descendants.
  • Search by firstname or lastname - Allows user to search for a specific member by entering a portion of the member's first or last name.
  • Data Validation - Prevents editing or adding a member with an invalid ParentId where no one has that ParentId.
  • Responsive UI - The app, including the descendant tree, updates immediately after an operation made by the user.

Tools and Technologies:

  • FastAPI - Handles backend data and requests.
  • MySQL - Stores people and their relationships.
  • Angular - Frontend framework for the user interface.
  • HTML / CSS - Structure and styling of the app.
  • TypeScript - Frontend logic and interactivity.

Useful resources used: FastAPI tutorial, Typescript tutorial
